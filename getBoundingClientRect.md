```js
element.getBoundingClientRect();
```

|    key    |        value가 의미하는 값         |
| :-------: | :--------------------------------: |
| top or y  | 화면 상단 부터 대상의 처음 위치 값 |
|  bottom   |  화면 상단 부터 대상의 끝 위치 값  |
| left or x | 화면 좌측 부터 대상의 처음 위치 값 |
|   right   |  화면 좌측 부터 대상의 끝 위치 값  |
|   width   |            대상의 길이             |
|  height   |            대상의 높이             |
